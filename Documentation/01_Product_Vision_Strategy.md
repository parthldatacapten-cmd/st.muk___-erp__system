## 3. Product Strategy

### 3.1 Core Philosophy

**"Configure, Don't Customize"**

Instead of writing custom code for each client:
1. **Pre-built Templates**: School, College, Coaching, University templates
2. **Workflow Engine**: Drag-drop workflow builder for approvals
3. **Rule Engine**: Configure ATKT rules, promotion criteria, fee structures
4. **Form Builder**: Custom forms for any data collection need
5. **Report Designer**: Build custom reports without SQL knowledge

### 3.2 Deployment Models

#### Model A: Self-Hosted (One-Time License)
```
Customer buys server → Installs EduCore → Configures → Goes live
License Fee: ₹2,00,000 - ₹20,00,000 (based on student capacity)
AMC: 15% of license fee/year (optional after Year 1)
```

#### Model B: Cloud SaaS (Subscription)
```
EduCore hosts on AWS/Azure → Customer accesses via web
Subscription: ₹20-₹50 per student/month
Minimum: ₹50,000/month
```

#### Model C: Hybrid
```
Core system self-hosted + Premium features cloud-based
License + Add-on subscriptions
```

## 6. Risk Assessment & Mitigation

### 6.1 Key Risks

| Risk | Probability | Impact | Mitigation |
|------|------------|--------|------------|
| Low tech adoption in Tier-3/4 | High | Medium | Offline mode, vernacular UI, training programs |
| Competition from established players | Medium | High | Focus on flexibility, pricing, customer service |
| Regulatory changes (education policy) | Medium | Medium | Modular design, quick update cycles |
| Data security concerns | Low | High | ISO 27001 certification, regular audits, encryption |
| Payment defaults from institutions | Medium | Medium | Advance payment model, usage-based licensing |

### 6.2 Compliance Requirements

- **Data Protection**: DPDP Act 2023 compliance
- **Financial**: GST invoicing, TDS deduction features
- **Education**: UDISE+, AISHE reporting formats
- **Accessibility**: WCAG 2.1 AA compliance
- **Security**: ISO 27001, regular penetration testing

## Appendix: Indian Education System Quick Reference

### School Levels
- **Pre-Primary**: Nursery, KG (Ages 3-5)
- **Primary**: Class 1-5 (Ages 6-10)
- **Middle**: Class 6-8 (Ages 11-13)
- **Secondary**: Class 9-10 (Ages 14-15) - Board Exams (SSC/CBSE 10th)
- **Higher Secondary**: Class 11-12 (Ages 16-17) - Board Exams (HSC/CBSE 12th)

### Higher Education Levels
- **Undergraduate**: 
  - Bachelor's Degree (BA, BSc, BCom, BTech, MBBS, etc.) - 3-4 years
  - Diploma/Polytechnic - 3 years
- **Postgraduate**:
  - Master's Degree (MA, MSc, MCom, MTech, MD/MS, etc.) - 2 years
  - PG Diploma - 1 year
- **Doctoral**: PhD - 3-5 years
- **Post-Doctoral**: Research Fellowships

### Major Education Boards
- **Central**: CBSE, CISCE (ICSE/ISC), NIOS
- **State**: Maharashtra State Board, Tamil Nadu State Board, etc. (28+ states)
- **International**: IB, IGCSE (Cambridge), American Curriculum

### Regulatory Bodies
- **Schools**: Ministry of Education, State Education Departments
- **Technical**: AICTE (Engineering), PCI (Pharmacy), BCI (Law)
- **Medical**: NMC (National Medical Commission)
- **Universities**: UGC (University Grants Commission)
- **Accreditation**: NAAC (Institutions), NBA (Programs)
